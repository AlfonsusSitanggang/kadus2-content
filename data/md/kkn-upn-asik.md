Log Hari Ke-14: KKN, Kode, dan Kenyataan di Desa Kecemen
1. Ekspektasi vs. Realitas
Sebagai seorang web developer, bayangan awal saya tentang KKN di Desa Kecemen adalah kontribusi digital yang keren. Saya berencana membuat Website Profil Desa yang canggih, lengkap dengan sistem pelayanan surat mandiri berbasis online dan pemetaan UMKM lokal.

Dalam kepala saya: Coding lancar, warga senang, nilai A di tangan.

Realitasnya? Baru hari pertama sosialisasi proker di balai desa, saya sudah dihadapkan pada kenyataan pahit.

2. Puncak Stres: Bug di Tengah Keterbatasan
"Mas, internet di dusun sebelah memang agak susah. Kadang kalau mau dapat sinyal bagus harus jalan ke dekat sawah atau naik ke lantai dua balai desa." Kalimat dari salah satu perangkat desa itu langsung membuat mental saya agak terguncang. Dan benar saja, seminggu terakhir ini rasanya seperti simulasi bertahan hidup di zaman purba bagi seorang anak IT:

Sinyal "Bisa Dicoba": Kecepatan internet di sini sering kali mentok di H+ atau Edge. Mau download dependency lewat npm install saja butuh waktu ditinggal tidur, itu pun sering kali timeout dan error.

Listrik Byar Pet: Sedang asyik debugging logika JavaScript yang rumit, tiba-tiba listrik padam karena hujan angin. Laptop mati, local server mati, dan pikiran saya ikut blank.

Benturan Kebutuhan: Saat saya fokus ingin menyelesaikan coding web, teman-teman sekelompok mengajak untuk ikut proker fisik seperti mengecat pos ronda, mengajar TPA, dan menanam bibit cabai. Kepala saya penuh dengan visualisasi Grid dan Flexbox, tapi tangan saya harus memegang cangkul.

3. Momen Paling Frustrasi
Kemarin adalah puncaknya. Saya mencoba mendemonstrasikan prototipe website di depan Kepala Desa dan para ketua RT.

Sinyal mendadak hilang, API lokal tidak merespons karena database belum tersinkronisasi sempurna akibat internet terputus semalam. Di depan, para tokoh masyarakat melihat layar proyektor dengan tatapan bingung.

Saya tersenyum canggung sambil menahan keringat dingin, mencoba menjelaskan konsep client-side error kepada warga yang sebenarnya lebih butuh solusi bagaimana cara mengajukan bantuan bansos dengan cepat. Stresnya tembus ke ubun-ubun.

4. Titik Balik: Menurunkan Ego
Semalam, sambil merenung di teras posko ditemani kopi hitam dan camilan keripik dari warga, saya sadar. Saya terlalu egois membawa standar industri teknologi ke desa yang bahkan literasi digitalnya masih mendasar.

Website keren dengan arsitektur cutting-edge tidak akan berguna kalau tidak ada warga yang bisa mengoperasikannya setelah kami pulang KKN.

Rencana Perubahan:
Sederhanakan Teknologi: Buang dulu framework yang terlalu berat. Kembalikan ke HTML, CSS, dan PHP/JS sederhana yang bisa berjalan mulus secara offline (lokal).

Fokus ke Edukasi: Daripada cuma pamer kode, waktu saya akan lebih banyak dipakai untuk melatih satu atau dua pemuda karang taruna di Desa Kecemen agar mereka tahu cara update konten webnya nanti.

Nikmati Prosesnya: Kalau internet mati, ya sudah. Waktunya tutup laptop, keluar posko, dan bantu teman-teman bikin plang penunjuk jalan.

KKN di Desa Kecemen ini ternyata bukan cuma tentang menguji seberapa jago saya menulis kode, tapi menguji seberapa waras dan adaptifnya saya menjadi seorang manusia di luar zona nyaman.
